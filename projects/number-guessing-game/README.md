# Number Guessing Game

A simple command-line number guessing game where the computer generates a random number and the player tries to guess it.

## Concepts practiced

- Random number generation
- User input and validation
- While loops
- Conditional logic
- Basic error handling

## Files in this project

- `README.md` - This file
- `main.py` - The main game logic

## How to run

```bash
python main.py
```

## Example commands

```bash
# Start the game
python main.py
```

The game will prompt you to guess a number between 1 and 100. Enter your guess and press Enter. The game will tell you if your guess is too high, too low, or correct.
