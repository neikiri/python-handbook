# CSV Sales Report

A simple tool that reads sales data from a CSV file and generates a summary report including total sales, sales by product, and average sale amounts.

## Concepts practiced

- CSV file reading
- Data aggregation
- Dictionary operations
- File I/O
- Basic statistics

## Files in this project

- `README.md` - This file
- `main.py` - The main application logic
- `sales.csv` - Sample sales data

## How to run

```bash
python main.py
```

## Example commands

```bash
# Run with default sales.csv
python main.py

# Run with a different CSV file
python main.py path/to/sales.csv
```
