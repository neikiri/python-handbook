# Mini Projects

This directory contains 8 mini projects to build your Python skills.

## Projects

1. **number-guessing-game** - CLI game with random number
2. **todo-cli** - Command-line todo list
3. **word-counter** - Count words/lines/chars in text
4. **csv-sales-report** - Read CSV, compute totals
5. **password-generator** - Generate secure passwords
6. **budget-tracker** - Track income and expenses
7. **log-analyzer** - Parse log files and summarize
8. **personal-notes-app** - Simple notes manager

## Structure

Each project has:
- `README.md` - Project description and requirements
- `main.py` - Main application code

## Running Projects

```bash
cd projects/number-guessing-game
python main.py
```

## Learning Path

1. Start with simpler projects (1-3)
2. Move to intermediate projects (4-6)
3. Try advanced projects (7-8)
4. Modify projects to add features
