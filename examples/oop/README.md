# OOP Examples

This folder contains examples demonstrating Object-Oriented Programming in Python:

- **basic_class.py**: Creating basic classes with attributes and methods
- **bank_account.py**: A practical bank account class
- **dataclass_example.py**: Using @dataclass for simpler classes
- **inheritance.py**: Class inheritance and method overriding
- **when_not_to_use_oop.py**: When to use functions instead of classes

## How to run

Each file is standalone. Run any example with:

```bash
python examples/oop/basic_class.py
```

Or from the `examples/oop` folder:

```bash
python basic_class.py
```
