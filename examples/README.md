# Examples

This directory contains runnable Python examples organized by topic.

## Structure

```
examples/
├── basics/       # Variables, types, operators, input
├── strings/      # String formatting and methods
├── collections/  # List, dict, set, tuple patterns
├── functions/    # Functions, args, kwargs, closures
├── oop/          # Classes, inheritance, dataclasses
├── files/        # Pathlib, JSON, CSV read/write
└── stdlib/       # datetime, random, re, collections
```

## How to Use

1. Navigate to a topic folder
2. Read the example file
3. Run it: `python example.py`
4. Modify it to experiment

## Running Examples

All examples are designed to run with Python 3.10+:

```bash
python examples/basics/hello.py
```
