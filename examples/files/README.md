# Files Examples

This folder contains examples demonstrating file operations in Python:

- **read_text_file.py**: Reading text files
- **write_text_file.py**: Writing text files
- **pathlib_paths.py**: Working with paths using pathlib
- **json_read_write.py**: Reading and writing JSON files
- **csv_read_write.py**: Reading and writing CSV files
- **simple_data_cleaning.py**: Simple data cleaning example
- **sample_notes.txt**: Sample text file for examples
- **sample_settings.json**: Sample JSON file for examples
- **sample_people.csv**: Sample CSV file for examples

## How to run

Each file is standalone. Run any example with:

```bash
python examples/files/read_text_file.py
```

Or from the `examples/files` folder:

```bash
python read_text_file.py
```
