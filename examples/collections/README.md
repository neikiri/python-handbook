# Collections Examples

This folder contains examples demonstrating Python collections:

- **lists.py**: Creating, modifying, and using lists
- **tuples.py**: Creating and using tuples (immutable lists)
- **sets.py**: Creating and using sets (unique values)
- **dictionaries.py**: Creating and using dictionaries (key-value pairs)
- **counting_items.py**: Counting items with dict and Counter
- **sorting_and_filtering.py**: Sorting and filtering collections

## How to run

Each file is standalone. Run any example with:

```bash
python examples/collections/lists.py
```

Or from the `examples/collections` folder:

```bash
python lists.py
```
