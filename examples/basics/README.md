# Basics Examples

This folder contains beginner-friendly examples demonstrating Python basics:

- **variables_and_types.py**: Variables, values, and built-in types (int, float, str, bool)
- **type_conversion.py**: Converting between types (str to int, int to float, etc.)
- **operators.py**: Arithmetic, comparison, and logical operators
- **input_and_output.py**: Using `input()` and `print()` with f-strings
- **basic_script_structure.py**: Simple script structure with `if __name__ == "__main__"`

## How to run

Each file is standalone. Run any example with:

```bash
python examples/basics/variables_and_types.py
```

Or from the `examples/basics` folder:

```bash
python variables_and_types.py
```
