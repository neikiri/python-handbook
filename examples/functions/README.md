# Functions Examples

This folder contains examples demonstrating Python functions:

- **basic_functions.py**: Creating and calling basic functions
- **parameters_and_return_values.py**: Function parameters and return values
- **default_arguments.py**: Default parameter values
- **args_kwargs.py**: *args and **kwargs for variable arguments
- **type_hints.py**: Type hints for function parameters and return values
- **simple_refactor.py**: Refactoring repeated code into functions

## How to run

Each file is standalone. Run any example with:

```bash
python examples/functions/basic_functions.py
```

Or from the `examples/functions` folder:

```bash
python basic_functions.py
```
