# Strings Examples

This folder contains examples demonstrating Python string handling:

- **string_basics.py**: Creating strings, indexing, and basic operations
- **string_methods.py**: Common string methods (upper, lower, strip, etc.)
- **slicing.py**: Extracting substrings with slicing
- **formatting.py**: f-strings and other formatting methods
- **text_cleanup.py**: Cleaning and normalizing user input

## How to run

Each file is standalone. Run any example with:

```bash
python examples/strings/string_basics.py
```

Or from the `examples/strings` folder:

```bash
python string_basics.py
```
