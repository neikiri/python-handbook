# Standard Library Examples

This folder contains examples demonstrating Python's standard library modules:

- **datetime_example.py**: Working with dates and times
- **random_example.py**: Generating random numbers
- **regex_example.py**: Using regular expressions
- **counter_example.py**: Using collections.Counter
- **itertools_example.py**: Using itertools for efficient iteration
- **argparse_example.py**: Parsing command-line arguments
- **logging_example.py**: Logging messages

## How to run

Each file is standalone. Run any example with:

```bash
python examples/stdlib/datetime_example.py
```

Or from the `examples/stdlib` folder:

```bash
python datetime_example.py
```
