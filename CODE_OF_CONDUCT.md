# Code of Conduct

## Our Pledge

We are committed to making participation in this project a welcoming and respectful experience for everyone, regardless of experience level, background, identity, or perspective.

## Expected Behavior

Examples of behavior that contributes to a positive environment:

- using welcoming and inclusive language
- being respectful of differing viewpoints
- giving constructive feedback
- accepting responsibility and apologizing when mistakes happen
- focusing on what is best for learners and contributors

## Unacceptable Behavior

Examples of unacceptable behavior include:

- harassment, insults, or personal attacks
- discriminatory language or behavior
- trolling or deliberately disruptive comments
- publishing private information without permission
- any conduct that would reasonably be considered inappropriate in a professional setting

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions that violate this Code of Conduct.

Repeated or serious violations may result in a temporary or permanent ban from participating in the project.

## Reporting

If you see behavior that violates this Code of Conduct, please report it by opening an issue or contacting the maintainers through the available GitHub channels.

Reports will be reviewed as respectfully and privately as possible.
