# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-27

### Added

- Complete Python handbook with 23 chapters covering basics to advanced topics
- 23 exercise sets with practical coding challenges
- 23 solution sets with detailed explanations and code
- 8 runnable example modules (basics, strings, collections, functions, OOP, files, stdlib)
- 8 mini projects (number guessing game, todo CLI, word counter, CSV sales report, password generator, budget tracker, log analyzer, personal notes app)
- 6 cheatsheets (syntax, built-ins, strings, collections, file I/O, standard library)
- 4 reference pages (built-in types, built-in exceptions, standard library modules, glossary)
- Comprehensive test suite with 94 tests covering all projects and examples
- MkDocs Material documentation site with full navigation
- GitHub Actions CI/CD workflows for testing and documentation
- Learning path guide with recommended reading order
- FAQ page for common beginner questions

### Features

- All code examples use Python 3.10+ syntax
- Type hints in intermediate and advanced chapters
- Standard library only (no third-party dependencies for core content)
- Beginner-friendly explanations from first principles
- Practical examples and projects
- Consistent chapter structure and terminology
- Accessible and inclusive content
