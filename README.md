A polished, beginner-friendly Python handbook that teaches Python from zero to practical independent programming.

## Overview

This repository contains:

- **Handbook** — 23 comprehensive chapters covering Python from basics to advanced topics
- **Exercises** — Practical coding challenges for each chapter
- **Solutions** — Detailed explanations and code solutions
- **Examples** — Runnable Python scripts demonstrating concepts
- **Mini Projects** — 8 practical projects to build your skills
- **Cheatsheets** — Quick reference guides for syntax and standard library
- **References** — In-depth documentation on built-ins and standard library

## Quick Start

### For Learners

1. Start with the [Learning Path](docs/learning-path.md) to understand the recommended reading order
2. Read chapters in sequence in the [Handbook](docs/handbook/)
3. Practice with [Exercises](docs/exercises/) and check [Solutions](docs/solutions/)
4. Build projects to reinforce your learning

### For Contributors

1. Fork the repository
2. Read [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines
3. Create a branch for your changes
4. Run tests: `pytest`
5. Build docs locally: `mkdocs serve`
